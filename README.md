# 🛒 Shopping List App (Jetpack Compose)

A simple **shopping list app** built with **Jetpack Compose**.  
You can add items, edit them, delete them, and mark them as purchased ✅.  
This project demonstrates **state management**, **state hoisting**, and **immutable data patterns** in Compose.

---

## ✨ Features
- ➕ Add new items with name and quantity  
- 📝 Edit existing items  
- ❌ Delete items  
- ✅ Mark items as purchased (with visual feedback)  
- 🎨 Modern UI with Material3 components  
- ♻️ Compose best practices (immutable data + `copy`)  

---
## 🛠 Tech Stack
- [Jetpack Compose](https://developer.android.com/jetpack/compose)  
- [Material3](https://m3.material.io/)  
- Kotlin
